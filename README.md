# Import Sort Style Antd

## Installation

```cmd
npm install -O github:airkro/import-sort-style-antd
```

## Usage

Add following settings to your `package.json`:

```json
{
  "importSort": {
    ".js, .jsx": {
      "parser": "babylon",
      "style": "antd"
    }
  }
}
```
